== arp.version.updated
*Severity*::
NOTICE
*Description*::
This message occurs when an Autonomous Ransomware Protection (ARP) version is updated on a node, whether by the automated update process or by user action.
*Corrective Action*::
(None).
*Syslog Message*::
Autonomous Ransomware Protection (ARP) version updated to %s.
*Parameters*::
*op* (STRING): Autonomous Ransomware Protection (ARP) version.
