== cam.counters.device.gone
*Severity*::
INFORMATIONAL
*Description*::
This message occurs when a device goes away when collecting I/O statistics.
*Corrective Action*::
(None).
*Syslog Message*::
Device %s is going away, current flags are 0x%x.
*Parameters*::
*device_name* (STRING): Device name.
*flags* (INTHEX): Device flags.
