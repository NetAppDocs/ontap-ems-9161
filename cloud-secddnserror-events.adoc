== cloud.SecdDNSError
*Severity*::
ERROR
*Description*::
This message occurs when this ONTAP Cloud instance fails to communicate with DNS/SECD for the cloud database endpoint.
*Corrective Action*::
Verify that DNS/SECD is set up correctly on this ONTAP Cloud instance and fix it if not.
*Syslog Message*::
This ONTAP Cloud instance fails to communicate with DNS/SECD for the cloud database endpoint "%s".
*Parameters*::
*db_endpoint* (STRING): Name of the cloud database endpoint associated with this ONTAP Cloud instance.
