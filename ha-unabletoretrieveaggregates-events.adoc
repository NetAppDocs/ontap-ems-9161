== ha.unableToRetrieveAggregates
*Severity*::
NOTICE
*Description*::
This message will be printed if, during boot, the node is unable to access its list of aggregates. This can happen if VLDB is out of quorum as the node boots.
*Corrective Action*::
(None).
*Syslog Message*::
This node was unable to retrieve the list of aggregates during boot. It will continue to retry until it is successful.
*Parameters*::
(None).
