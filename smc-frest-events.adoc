== smc.frest.snap.delete.fail
*Severity*::
ERROR
*Description*::
This message occurs when a SnapMirror file restore operation fails to delete a local file restore snapshot on the destination volume.
*Corrective Action*::
Manually delete snapshot by issuing snapshot delete command.
*Syslog Message*::
Unable to delete snapshot '%s' from volume '%s'.
*Parameters*::
*snapname* (STRING): Snapshot that was not deleted.
*dstpath* (STRING): Path name of the destination volume to which restore was issued.
