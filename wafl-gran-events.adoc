== wafl.gran.data.adv.not.on
*Severity*::
ERROR
*Description*::
This message occurs when a SnapMirror transfer fails because a snapshot that contains multipart inodes is transferred to a destination volume that does not support advanced granular data. Multipart inodes are created on volumes with granular data enabled.
*Corrective Action*::
Enable granular data on the destination volume using the command <granular-data advanced>".
*Syslog Message*::
SnapMirror transfer failed because the transfer snapshot contains multipart inodes and the destination volume, "%s", does not support them.
*Parameters*::
*volName* (STRING): Volume name.
